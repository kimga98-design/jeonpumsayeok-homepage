# 위험신호 관찰 체크리스트

> [[순장-현장돌봄]]이 성도를 관찰할 때 사용하는 참고 도구
> ⚠️ 임상 진단 도구 아님 — 전문 진단은 반드시 전문의에게

---

## 🌧 기분·감정 신호 → [[우울장애]] · [[양극성장애]]
- [ ] 2주 이상 지속적으로 우울하거나 무기력
- [ ] 예전에 좋아하던 활동에 흥미 상실
- [ ] 감정 기복 심함, 이유 없이 울거나 화냄
- [ ] 자신을 무가치하다·죄책감 많다고 표현

## 😰 불안·긴장 신호 → [[공황장애]] · [[강박장애]]
- [ ] 사소한 일에도 지나치게 걱정·두려워함
- [ ] 갑자기 심장 두근거림·숨막힘 호소
- [ ] 특정 장소·상황을 극도로 회피
- [ ] 같은 행동 반복, 특정 생각에서 못 벗어남

## 😴 신체·생활 신호 → [[우울장애]] · [[PTSD]]
- [ ] 수면 현저히 줄거나 지나치게 많이 잠
- [ ] 급격한 체중 변화
- [ ] 집중력 저하, 대화 중 자주 멍해짐
- [ ] 원인 불명의 두통·소화불량 자주 호소

## 🤝 관계·공동체 신호 → [[PTSD]] · [[조현병]]
- [ ] 모임 반복 결석, 연락 회피
- [ ] 주변 관계가 눈에 띄게 단절
- [ ] 평소와 다른 이상한 말·행동 관찰

## 🚨 즉각 연락 필요 — 하나라도 해당 시 [[위급연락망]] 바로

관련 질환: [[자해-자살위기]]
- [ ] "죽고 싶다 / 사라지고 싶다" 말 반복
- [ ] 자해 흔적·위험 물건(칼, 약물) 발견
- [ ] 수일째 연락 완전 두절
- [ ] 타인을 해칠 것이라는 말·행동
- [ ] 갑자기 소중한 물건 나눔·작별 인사
- [ ] 현실과 동떨어진 이야기 진지하게 반복

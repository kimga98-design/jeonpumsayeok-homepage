# 도움 요청 4단계 절차

> 혼자 판단하지 않고, 단계별로 안전하게 연결한다.

---

## 단계별 흐름

```
1단계 → 2단계 → 3단계 → 4단계
순장    정품사역자  정품사역팀   전문기관
감지    연결       자문        치료
```

### Step 1 — [[순장-현장돌봄]] (감지·경청)
성도의 어려움 관찰 → [[위험신호-관찰]] 체크
→ 즉시: [[정품사역자-연결자]]에게 알림

### Step 2 — [[정품사역자-연결자]] (연결)
상황 파악 → [[정품사역팀-지원체계]]와 협력
→ 적절한 자원 방향 논의

### Step 3 — [[정품사역팀-지원체계]] + 교역자 (자원 조율)
외부 전문기관 네트워크 연결
정신건강복지센터 → 국립병원 → 지역 전문의 순

### 응급 — 지체 없이 [[위급연락망]]
즉각 위험신호 발견 시 단계 무시하고 바로

---

## 질환별 연결 방향
| 질환 | 우선 연결 |
|---|---|
| [[조현병]] · [[양극성장애]] | 정신건강의학과 전문의 |
| [[우울장애]] · [[PTSD]] | 상담센터 + 정품사역팀 |
| [[공황장애]] · [[강박장애]] | CBT 전문 상담 |
| [[자해-자살위기]] | 즉시 [[위급연락망]] |
| [[ADHD]] | 소아청소년/성인 전문 클리닉 |

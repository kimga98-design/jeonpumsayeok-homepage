# 협력 돌봄 원칙

> 정품사역은 **절대 혼자 감당하지 않는다**. 협력이 안전을 만든다.

---

## 왜 혼자 하면 안 되나
- 판단 오류 위험 (관계가 가까울수록)
- 소진(번아웃) 위험
- 책임 소재 불명확 → 성도 보호 취약

## 협력 구조
```
[[순장-현장돌봄]] (현장의 눈)
    ↕
[[정품사역자-연결자]] (전문적 연결)
    ↕
[[정품사역팀-지원체계]] (자원 조율)
    ↕
교역자 (영적·공동체적 총괄)
```

## 실천
- 감지하면 혼자 해결 시도 전 → 즉시 [[정품사역자-연결자]]에게 공유
- 판단이 필요할 때 → [[정품사역팀-지원체계]] 자문
- 위기 상황 → [[위급연락망]] + 교역자 동행

# 🌳 어원: **aud** (듣다 (to hear))

**aud** 는 '듣다(hear)'를 의미하는 라틴어 어근입니다.

| 어근 + 접미사 | = 단어 | 뜻 |
|--------------|--------|----|
| aud + ience | = audience | 청중 (듣는 사람들) |
| aud + ible | = audible | 들을 수 있는 |
| aud + io | = audio | 음성, 오디오 |
| aud + it | = audit | 감사하다 (들어보다) |
| aud + itorium | = auditorium | 강당 (듣는 공간) |

💡 **auditorium(강당)** 은 '소리를 듣는 공간'이라는 뜻이에요!
